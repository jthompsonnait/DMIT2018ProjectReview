Access Control Summary

The Admin role has full access to all areas and all features.
In each area below, Managers can see the restricted features, while Staff cannot.

Purchase Orders (PO)

Restricted Feature: Place Order

  • ✅ Admin – Full access

  • ✅ Manager (Shop Manager) – Can place orders

  • ❌ Staff (Store Staff) – Cannot place orders

Receiving

Restricted Feature: Force Close

  • ✅ Admin – Full access

  • ✅ Manager (Shop Manager) – Can force close

  • ❌ Staff (Store Staff) – Cannot force close

Sales

Restricted Feature: Sales Return

  • ✅ Admin – Full access

  • ✅ Manager (Sales Manager) – Can process returns

  • ❌ Staff (Salesperson) – Cannot process returns

Service

Restricted Feature: Coupon Area

  • ✅ Admin – Full access

  • ✅ Manager (Parts Manager) – Can access coupons

  • ❌ Staff (Mechanic) – Cannot access coupons
