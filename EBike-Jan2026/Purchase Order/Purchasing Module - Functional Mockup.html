Place Requirements

Place is only enabled when ALL of these are true: - Order has been saved (has a PO#) - Order is not already placed - Every line has QTO >= 1 - Every line has Price >= 0 (zero allowed for valid business reasons such as marketing materials or vendor compensation) - At least one line item exists

What Place does: Sets OrderDate = today, updates QOO for each part, locks the entire order.

D2 Security: Place is restricted to Shop Manager role only.

What the System Must Never Do

  • Never auto-save a suggested order вҖ” user must explicitly press Save
  • Never apply suggestions to an existing unplaced order
  • Never allow a placed order to be modified or deleted
  • Never auto-fill QTO for manually added or re-added items (always 0)
  • Never allow the same part to appear twice in one order
  • Never allow QOH to go negative
  • Never save an order where any line has QTO = 0
